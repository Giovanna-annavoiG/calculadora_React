# 🧮 Calculadora React + Math.js

Uma calculadora funcional e responsiva desenvolvida como projeto de estudo para praticar os conceitos fundamentais de React.js, manipulação de estados e integração com bibliotecas externas.

## 🚀 Tecnologias Utilizadas
* **React.js**: Biblioteca principal para a interface.
* **Math.js**: Utilizada para garantir precisão e segurança nos cálculos matemáticos.
* **CSS3**: Estilização personalizada focada em User Experience (UX).

## ✨ Funcionalidades
- Operações básicas (Soma, Subtração, Multiplicação e Divisão).
- Tratamento de expressões complexas via Math.js.
- Interface Responsiva (funciona no celular e no PC).
- Botão de limpar (C) e deletar caractere.

## 🛠️ Como rodar o projeto localmente
1. Clone este repositório:
   `git clone https://github.com/Giovanna-annavoiG/calculadora_React.git`
2. Entre na pasta:
   `cd calculadora_React`
3. Instale as dependências:
   `npm install`
4. Inicie o projeto:
   `npm start`
